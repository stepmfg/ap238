Working Draft ISO 10303-238 Edition 2

Annex E
(normative)

Information object registration

E.1 Document identification

In order to provide for unambiguous identification of an information object in an open system, the object identifier

  • { iso standard 10303 part(238) version(2) }

is assigned to this part of ISO 10303. The meaning of this value is defined in ISO/IEC 8824-1, and is described in ISO 10303-1.

E.2 Schema identification

In order to provide for unambiguous identification of the integrated_cnc_schema in an open information system, the object identifier

  • { iso standard 10303 part(238) version(1) schema(1) integrated-cnc-schema(2) }

is assigned to the integrated-cnc-schema schema (see Annex A). The meaning of this value is defined in ISO/IEC 8824-1, and is described in ISO 10303-1.
