Annex G
(informative)

Application reference model

This annex provides the application reference model for this part of ISO 10303. The application reference model is a graphical representation of the structure and constraints of the application objects specified in clause 4, including the application objects referenced from ISO 14649-10, ISO 14649-11, ISO 14649-12, ISO 14649-111 and ISO 14649-121. The graphical form of the application reference model is presented in EXPRESS-G. The application reference model is independent from any implementation method. EXPRESS-G is defined in Annex D of ISO 10303-11:1994.
