ISO 10303-238

Annex C
(normative)

Implementation method-specific requirements

C.1 General

The implementation method defines what types of exchange behaviour are required with respect to this document. Conformance to this document shall be realized in an exchange structure described by either ISO 10303-21 or ISO 10303-28.

C.2 Requirements specific to ISO 10303-21

If the implementation method is ISO 10303-21, the file format shall be encoded according to the syntax and EXPRESS language mapping defined in ISO 10303-21 and in the AIM expanded listing defined in Annex A of this document. The header of the exchange structure shall identify use of this document by the schema name 'MODEL_BASED_INTEGRATED_MANUFACTURING_SCHEMA'.

C.3 Requirements specific to ISO 10303-28

If the implementation method is ISO 10303-28, the file format shall be encoded according to the syntax and EXPRESS language mapping defined in ISO 10303-28 and in the AIM expanded listing defined in Annex A of this document. The ISO 10303-28 configuration shown below shall be used.

<?xml version="1.0"?>
<configuration 
   xmlns="urn:oid:1.0.10303.28.2.1.2"
   xmlns:xs="http://www.w3.org/2001/XMLSchema">

   <schema targetNamespace="urn:oid:1.0.10303.238.2.0.1" />
   <option exp-attribute="attribute-content" tagless="true" />
   <uosElement name="iso_10303_28_terse" >
     <add_attribute name="schema" type="xs:NMTOKEN" usage="required" />
   </uosElement>
</configuration>

The iso_10303_28_terse element shall identify the use of this document by the namespace 'urn:oid:1.0.10303.238.2.0.1' and the 'schema' attribute value of 'model_based_integrated_manufacturing_schema'.

EXAMPLE The ISO 10303-28 configuration results in instance data written using the "attribute-content" encoding enclosed in the iso_10303_28_terse UOS element, as shown in the fragment below.

<?xml version="1.0"?>
<iso_10303_28_terse
    xmlns="urn:oid:1.0.10303.238.2.0.1"
    xmlns:exp="urn:oid:1.0.10303.28.2.1.1"
    xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
    schema="model_based_integrated_manufacturing_schema">

   <exp:header>
     <exp:name>dataset</exp:name>
     <exp:preprocessor_version>stepnc writer</exp:preprocessor_version>
   </exp:header>

  <!-- instance data -->
  <Axis2_placement_3d id="id1" Name="" Location="id2" Axis="id3" Ref_direction="id4"/>
  <Cartesian_point id="id2" Name="loc" Coordinates="3.5 -3.5 -4.16875"/>
  <Direction id="id3" Name="Z direction" Direction_ratios="0 0 1"/>
  <Direction id="id4" Name="X direction" Direction_ratios="1 0 0"/>
</iso_10303_28_terse>
