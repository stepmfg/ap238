ISO 10303-238

Annex E
(normative)

Information object registration

E.1 Document identification

In order to provide for unambiguous identification of an information object in an open system, the object identifier

  • { iso standard 10303 part(238) version(2) }

is assigned to this document. The meaning of this value is defined in ISO/IEC 8824-1, and is described in ISO 10303-1.

E.2 Schema identification

In order to provide for unambiguous identification of the model_based_integrated_manufacturing_schema in an open information system, the object identifier

  • { iso standard 10303 part(238) version(2) schema(1) model-based-integrated-manufacturing-schema(2) }

is assigned to the model-based-integrated-manufacturing-schema schema (see Annex A). The meaning of this value is defined in ISO/IEC 8824-1, and is described in ISO 10303-1.
