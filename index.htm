ABSTRACT: This document specifies an application interpreted model (AIM) based on the application reference model for machining defined by ISO 14649-10, ISO 14649-11, ISO 14649-12, ISO 14649-17, ISO 14649-111 and ISO 14649-121, augmented with ISO 10303 product geometry, geometric dimensioning and tolerancing and product data management information.

KEYWORDS: data exchange, computerized numerical controler (CNC), machine tool, machining, milling, turning, additive manufacturing, GD&T, STEP-NC.
COMMENTS TO READER:

This is the second edition of AP238. The document includes changes accumulated 2007-2019 from first edition deployment experience, as detailed in TC184/SC4/WG15N2. It has been harmonized with the latest edition of AP242 via shared references to the SMRLv8 1000-series parts of ISO 10303. Application objects for GD&T, feature, management data, and machine kinematics are now referenced in the SMRL rather than defined locally. Additive manufacturing process is now supported based on machine actions defined by ISO 14649-17.

The new Annex K change log describes the changes in detail.
