3 Terms and definitions

For the purposes of this document, the following terms and definitions apply.

3.1 Terms defined in ISO 1101

  • dimension;
  • tolerance.

3.2 Terms defined in ISO 2806

  • computerized numerical control (CNC);
  • numerical control (NC);
  • tool path.

3.3 Terms defined in ISO 5459

  • datum.

3.4 Terms defined in ISO 10303-1

  • application;
  • application activity model (AAM);
  • application interpreted model (AIM);
  • application object;
  • application protocol (AP);
  • application reference model (ARM);
  • conformance testing;
  • implementation method;
  • integrated resource;
  • protocol implementation conformance statement (PICS);
  • product;
  • product data;
  • unit of functionality (UoF).

3.5 Terms defined in ISO 10303-11

  • complex entity instance;
  • simple entity instance.

3.6 Terms defined in ISO 14649-1

  • executable;
  • machining operation;
  • NC function;
  • program structure;
  • project;
  • workingstep;
  • workplan.

3.7 Term defined in ISO 14649-10

  • freeform machining.

3.8 Terms defined in ISO 14649-11

  • finishing;
  • roughing.
